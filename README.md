1. Buatlah program yang bisa mengkonversi nilai angka yang di inputkan user menjadi
nilai huruf, dengan kondisi:
- nilai 85 - 100, mendapat A.

- nilai 40 - 69 mendapat C.

inputkan bilangan 2= 7
inputkan bilangan 3= 22
